## 📊 Overview

This project presents a comprehensive data-driven analysis of alcohol consumption patterns in India and their relationship with mental health indicators such as **stress, depression, tobacco usage, and suicide rates**.

By integrating multiple real-world datasets and visualizing them through Power BI, the project uncovers hidden behavioral patterns and supports **data-driven public health decision-making**.

---

## 🌍 Why This Project Matters

Mental health and substance abuse are critical public health challenges in India. However, insights are often buried in raw datasets and reports.

This project bridges that gap by:

* Converting complex datasets into **interactive visual insights**
* Identifying **high-risk populations and regions**
* Supporting **policy-making and awareness programs**

---

## 🎯 Objectives

* Analyze alcohol consumption trends across Indian states (2018–2024)
* Identify correlations with mental health indicators
* Detect high-risk demographic groups
* Build an interactive dashboard for insight generation

## 📂 Datasets Used

* NFHS-5 State-Level Indicators (2018–2021)
* Suicide-by-Cause Dataset (2018–2023)
* Stress Rate Dataset (2014–2023)
* Smoking & Lifestyle Dataset
* Health Department Reports

## ⚙️ Methodology

### 🔹 Data Collection

* Aggregated multiple public datasets related to health and lifestyle

### 🔹 Data Cleaning

* Removed duplicates and null values
* Standardized state names and formats
* Converted data types
* Created age-group categories

### 🔹 Data Modeling

* Built relationships between datasets
* Created DAX measures:

  * `TotalDeaths`
  * `AlcoholDeaths`
  * `Stress Index`

### 🔹 Visualization

* Developed interactive dashboards using Power BI:

  * State-wise maps
  * Age-group analysis
  * Trend charts
  * Correlation visuals

## 🔍 Key Insights

* 📍 Higher alcohol consumption observed in **northeastern and southern states**
* 👥 Age group **25–44** contributes the largest share of consumption
* 📈 Strong correlation between:

  * Alcohol consumption
  * Stress levels
  * Tobacco usage
  * Suicide rates
* 💀 Alcohol-related deaths are highest in major states like Maharashtra and Tamil Nadu
* 🧠 Increased mental health issues observed in high-consumption groups

👉 These insights can help design **targeted intervention programs and awareness campaigns**.
